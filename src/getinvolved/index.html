Get Involved

Wondering how to get involved with Decred? We have a whole list of projects for you to contribute to. Get familiar with the project your interested in, hack on it, and send us a pull request!

Decred related GitHub repositories

For maintaining a list of GitHub repositories related to the Decred project. If you have a repository to add, open a pull request adding the details.

Software

Selected repositories housing important Decred software. Issues are typically used to coordinate work.

  • decrediton - GUI wallet
  • dcrd and dcrwallet - Decred daemons
  • politeia and politeiagui - Politeia back and front end
  • dcrdata - block explorer
  • dcrweb - Decred.org
  • dcps - Decred Change Proposals

    • Not (primarily) Software

    decred/dcrdocs - main documentation site at docs.decred.org. The documents themselves live in folders corresponding to their sections within the /docs/ folder. The #DCR documentation channel is where people who work on dcrdocs discuss it. GitHub Issues are used to make suggestions about changes, like new pages or re-writing or re-structuring content. Pull Requests to change content should typically be in relation to some open issue and/or discussion in #DCR documentation.

    decred/dcrdesign -

    raedahgroup/video-production - production of Decred video explainers, including scripts and storyboards for work in production. These videos go through stages of production and feedback is typically sought in #DCR Writers at certain milestones.

    xaur/decred-news - monthly Decred Journal maintained by @bee. Seeks review and contributions from other community members in #DCR documentation around the start of each month, with the journal being published around the 6th.

    heyvj/decred-events - @vj maintains a set of guidelines and best practices for attending events and hosting meetups. Community members should submit logs of events where Decred had a presence as well as information on upcoming events.

    richardred0x - has repositories for writing (feedback usually sought before publication), analysis (data and scripts available, welcomes improvements or extensions to code), and an older repo with Dash data and analysis.

    dezryth - @dezryth maintains repositories for writing and code.

    karamble/dcrstatus - dashboard for Decred Command Line Interface users.
